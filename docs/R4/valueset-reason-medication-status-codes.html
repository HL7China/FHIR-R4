Summary

Version:4.0.0
Name:SNOMEDCTDrugTherapyStatusCodes
Title:SNOMED CT Drug Therapy Status codes
Definition:

This value set includes some taken and not taken reason codes from SNOMED CT - provided as an exemplar

  • Resource: MedicationStatement.statusReason (CodeableConcept / Example)

4.4.1.620.1 Content Logical Definition

This value set includes codes from the following code systems:

  • Include codes from http://snomed.info/sct where concept is-a 266710000 (Drugs not taken/completed)
  • Include codes from http://snomed.info/sct where concept is-a 410684002 (Drug therapy status)
