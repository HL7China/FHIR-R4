11.1.5 Resource MedicationRequest - Examples

Pharmacy Work GroupMaturity Level: N/AStandards Status: InformativeSecurity Category: Patient Compartments: Encounter, Patient, Practitioner

ORAL DOSAGE FORMS

The following examples are MedicationRequest resources for Oral Dosage Forms that reference Medication Resources for generic formulations:

  • Active Request with link to encounter, reasonCode, note multiple dosage lines, dispenseRequest and substitution - Azithromycin. Also includes non-clinical supportingInformation
  • Request with as needed (PRN) dosage with pre-condition - Oxycodone - active with reasonCode, note, dispenseRequest and substitution
  • Request for Tapering Dose - Prednisone - active with note and multiple DosageInstructions with boundsPeriod
  • Request for Oral Drops - Nystatin Suspension - completed - with Dispense Request
  • Request with Dosage with a Pre-Condition - Alprazolam - dosageInstructions includes boundsPeriod with periodMax and asNeededCodeableConcept
  • Request for a Dosage that needs to be dispensed as different strength tablets - Warfarin - active

The following examples are MedicationRequest resources for Oral Dosage Forms that reference Medication Resources for manufactured products:

  • Oral Chemotherapy Request - Myleran - completed - with reasonCode
  • Request with as needed (PRN) dosage - Percocet - completed with link to encounter, dispenseRequest and substitution
  • Request for as needed (PRN) dosage - Vicodin - completed with dosageInstruction with additionalInstruction (coded) and as needed with reason, dispenseRequest and substitution
  • Oral Chemotherapy Request - Capecitabine - active medication with dosing by Body Surface Area (BSA) (mg/m2)
  • Request for Over the Counter Medication - Tylenol PM - with dosage range as needed with pre-condition for use
  • Request for a medication that includes the dosage of a prescription in text
  • Fully populated example of a MedicationRequest - Chlorthalidone - active - with link to encounter, reasonReference, note, dispenseRequest, substitution and eventHistory

The following examples are MedicationRequest resources for Oral Dosage Forms that include a Medication Code

  • Liquid Medication - Phenytoin with link to prior prescription
  • Request with a single dosage line - Azithromycin - completed - with link to prior prescription and reason code
  • Request - Levothyroxine - completed with reasonCode, dosageInstructions and dispenseRequest

INJECTABLE DOSAGE FORMS

The following examples of MedicationRequest resources for Injectable Dosage Forms that reference a Medication Resource for a generic formulation

  • Request for an injectable medication - Lorazepam with Dosage Rate (as rateRange) - active - with link to encounter, reasonCode and dosageInstruction

The following examples of MedicationRequest resources for Injectable Dosage Forms that reference a Medication Resource for a manufactured product:

  • Request for Intravenous Chemotherapy (inpatient Request) - Brentuximab Vedotin - completed
  • Request for Intravenous Chemotherapy (inpatient Request) - Alemtuzumab - completed with multiple dosageInstruction lines
  • Request for Intravenous Antibiotic - Vancomycin - active - with link to encounter, reasonCode, and note
  • Request for Intravenous Antibiotic with a Dosage Rate (as RateQuantity) - Piperacillin - completed with reason code
  • Request for Subcutaneous injection with multiple dosage lines - Insulin - Novolog - active - includes reasonCode and note. Dosage includes additionalInstructions as text

The following examples of MedicationRequest resources for Injectable Dosage Forms that include a Medication Code:

  • Request for Subcutaneous injection - Insulin - Lantus - completed, with reason code and dispense request

The following examples of MedicationRequest resources for Injectable Dosage Forms for Injectable Dosage Forms where the medication is a compounded (aka extemporaneous or magistral) product:

  • Request for Intravenous Admixture with Dosage Rate (as rateRatio) - Potassium Chloride in D5W - completed with reason code
  • Request for TPN Solution with Dosage Rate (as rateRatio) - completed

MISCELLAENOUS DOSAGE FORMS

The following examples of MedicationRequest resources including miscellaneous dosage forms:

  • Request using Medication Code for Rectal Suppository - Acetaminophen - completed with reasonCode, dispenseRequest and substitution
  • Request using Medication Code for Topical Ointment - Nystatin - on hold and with dispense request
  • Request using Medication Code for Inhaler - Proventil - on hold with dispenseRequest and substitution
  • Request using Medication Code for Patch - Fentanyl - active, with dispense request
  • Medication Resource Reference - Nasal Spray - Nasonex
  • Request referencing Medication Resource for Compounded Product - HC 1%, Salicylic Acid 5% in Glaxal Base - on hold with dispenseRequest
  • Request referencing Medication Resource for Eye Drops - Timoptic - active - with link to encounter, dispenseRequest and substitution flag
  • Request referencing Medication Resource for a Kit that contains 2 different dosage forms - Vagistat

MISCELLAENOUS DOSAGE INSTRUCTIONS

The following examples of MedicationRequest resources including miscellaneous dosage instructions:

  • Request that includes a dosage instruction for a medication that is intended to be given once only as single dose
  • Request for a medication that is intended to be given in the morning
  • Request that includes a dosage instruction for a medication that is given on a specific day of the week
  • Request that includes a dosage instruction for a medication that is given at a specific time of the day
  • Request with as needed (PRN) dosage with pre-condition - Oxycodone - active with reasonCode, note, dispenseRequest and substitution
  • Request for Tapering Dose - Prednisone - active with note and multiple DosageInstructions with boundsPeriod
  • Request with Dosage with a Pre-Condition - Alprazolam - dosageInstructions includes boundsPeriod with periodMax and asNeededCodeableConcept
  • Request for an injectable medication - Lorazepam with Dosage Rate (as rateRange) - active - with link to encounter, reasonCode and dosageInstruction
  • Request for Intravenous Admixture with Dosage Rate (as rateRatio) - Potassium Chloride in D5W - completed with reason code
  • Request for Intravenous Dopamine - Weight based calculation with rateQuantity
  • Request for Intravenous Admixture Dopamine - with rateQuantity
  • Request for Intravenous Admixture Dopamine - with doseQuantity and rateQuantity
  • Request for Subcutaneous injection with multiple dosage lines - Insulin - Novolog - active - includes reasonCode and note. Dosage includes additionalInstructions as text
  • Request for Intravenous Antibiotic with a Dosage Rate (as RateQuantity) - Piperacillin - completed with reason code
  • Oral Chemotherapy Request - Capecitabine - active medication with dosing by Body Surface Area (BSA) (mg/m2)
  • Request for as needed (PRN) dosage - Vicodin - completed with dosageInstruction with additionalInstruction (coded) and as needed with reason, dispenseRequest and substitution
