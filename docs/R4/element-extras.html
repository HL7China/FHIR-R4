2.28.2 Element Extensions (can be used anywhere)

工作组: FHIR Infrastructure 成熟度: N/A标准状态: Informative

The extensions defined here can be used on any element. For background information, see Extensibility. Additional extensions may be found in published Implementation Guides , or in the Conformance resource registry .
