4.0 Terminology Module术语组件

4.0.1 Introduction简介

术语组件提供了FHIR资源、操作、编码数据类型和外部定义的标准以及FHIR定义的术语的概述和指南，这些术语用于表示和解释FHIR核心规范和配置中的编码结构化数据。 总的来说，上述能力用于提供功能性术语服务，以支持在整个规范中使用FHIR资源中的编码数据，在FHIR规范的其他组件也会对其有所描述。

4.0.4 Common Use Cases常见用例

=======

4.0.4 Common Use Cases

>>>>>>> c4c8d4713ee76294acf1833dd24b4aa4ea2ba3a9

  • Create or reference a code system创建或引用代码系统
  • Create or reference a value set创建或引用值集
  • Record data using pre-coordinated codes使用预先协调的代码记录数据
  • Record data using post-coordinated expressions使用后协调表达式记录数据
  • Record data using multiple local or standard code systems (translations)使用多个本地或标准代码系统（翻译）记录数据
  • Expand a value set展开值集
  • Validate a code验证代码
  • Look up a display term for a code查找代码的显示术语
  • Translate a code from one value set to another将代码从一个值集转换为另一个值集
  • Maintain a client-side transitive closure table on subsumption relationships在包含关系上维护客户端传递闭包表
  • Test subsumption between concepts概念间的测试假设
  • For a set of property/concept pairs, return the set of concepts for the requested properties对于一组属性/概念，返回所请求属性的概念集。
  • Map data between different terminologies在不同术语之间映射数据
  • Declare the capabilities of a terminology service声明术语服务的功能
