0 欢迎使用 FHIR®

FHIR是HL7®出版的医疗保健数据交换标准。

第一次来这里？
请参阅摘要, 开发人员入门, 临床人员入门 , 和 架构师入门, 然后查看 FHIR 概述 / 路线图 & 时间表. 另请参阅 开放许可 （不要错过完整的 目录表 和 社区贡献。 当然，你也可以 搜索本规范)。
