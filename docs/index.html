HL7 FHIR标准是一个软件数据交互标准。它由国际HL7组织发布的一个平台式标准规范，它定义了一系列在进行医疗健康活动中所需的“平台能力”。它能灵活支持不同区域不同医疗机构的不同场景，包括所有的国家和地区。

本网站是HL7 FHIR 官方网站的中文版，目标是助推下一代医疗数据交换标准在中国的实施。提供包括HL7 FHIR标准规范的中文翻译、FHIR资源的中国本地化、FHIR实施指南、FHIR术语服务、FHIR测试服务器、开源FHIR组件与工具、FHIR培训、FHIR项目实施咨询服务等。
